Voice Setup — ElevenLabs

The demo uses a Cloudflare Worker to call ElevenLabs securely.
Two steps — takes 5 minutes total.

1
Deploy the Cloudflare Worker
Go to dash.cloudflare.com → Workers & Pages → Create → paste the Worker file.
Name it decisight-tts → Add secret EL_KEY = your ElevenLabs key → Deploy.
2
Upload demo to GitHub Pages
Push demo-iso.html → demo/index.html on GitHub.
The Worker URL is already set in the code. Done.
🔒 Your ElevenLabs key stays in Cloudflare as a secret — never in the browser or HTML source.
